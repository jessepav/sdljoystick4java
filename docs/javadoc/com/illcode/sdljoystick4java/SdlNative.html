com.illcode.sdljoystick4java

Class SdlNative

  • java.lang.Object
    • com.illcode.sdljoystick4java.SdlNative

  • public final class SdlNative
extends java.lang.Object
    Direct mappings for JNI methods.

    Note that you will need to load the native shared libaries before calling any of the native methods in this class. Typically you can do this via loadNative(boolean), but if you have your own system in place, we don't force any conventions on you by putting the System.loadLibrary() calls in a static {} block.

      • joystickGetBall

        public static int joystickGetBall(long joystickPtr,
                  int ball,
                  int[] deltas)
        Get the ball axis change since the last poll. Trackballs can only return relative motion since the last call to this method.
        Parameters:
        joystickPtr - joystick pointer value
        ball - the ball index to query; ball indices start at index 0
        deltas - the difference in the axis position since the last poll, with index 0 storing dx and index 1 storing dy; thus, the passed array must have at least 2 elements -- no check will be performed, and memory corruption will likely result if you pass a shorter array.
        Returns:
        0 on success or a negative error code on failure
