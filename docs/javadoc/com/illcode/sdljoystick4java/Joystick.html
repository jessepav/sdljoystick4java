com.illcode.sdljoystick4java

Class Joystick

  • java.lang.Object
    • com.illcode.sdljoystick4java.Joystick

  • public class Joystick
extends java.lang.Object
    A wrapper class for the native joystick methods found in SdlNative.

    Prior to using this class, you'll need to call SdlNative.initJoysticks().

      • setTransitionDetectionEnabled

        public void setTransitionDetectionEnabled(boolean transitionDetectionEnabled)
        Enable or disable support for button transition events. These are exposed through the buttonPressed(int) and buttonReleased(int) methods. The usual button polling method, getButton(int), reports if the button was pressed at the time of the last call to update(boolean). The button transition methods, in comparison, report if the button was pressed (or released) in the most recent call to update(), and not pressed (or released) at the time of the previous update().
        Parameters:
        transitionDetectionEnabled - transition detection state
